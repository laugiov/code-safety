# Getting Started

This section will help you set up and run Taint Analysis Masterclass on your local machine.

## Overview

Taint Analysis Masterclass consists of several components:

1. **VulnShop** - A vulnerable Django application (the analysis target)
2. **Analysis Configurations** - Pysa, CodeQL, and Semgrep setups
3. **Benchmark Suite** - Tools to measure detection accuracy
4. **Documentation** - This documentation site
